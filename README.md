# Proyecto de Digitalización de Registros

## Objetivo

El presente proyecto busca brindar una forma más eficiente y accesible
de registrar el mantenimiento realizado en la ciudad de Fernández y
alrededores. Para ello, se procedió a la digitalización completa de los
registros físicos y su posterior estructuración en una base de datos
digital.\
El objetivo final es agilizar el proceso de carga, consulta y análisis
de los trabajos realizados, permitiendo una gestión más clara, rápida y
comprensible.

------------------------------------------------------------------------

## Contexto

El Área de Electricidad de la Municipalidad de Fernández gestiona los
reclamos de los vecinos y conserva un registro manuscrito de sus
actividades.\
La entidad cuenta con una base de datos pública registrada en papel,
escrita a mano, con aproximadamente setecientos registros no
estructurados.

Debido a que los datos son de carácter público y no contienen
información sensible o personal de los vecinos, no fue necesaria la
anonimización durante el proceso de digitalización.

------------------------------------------------------------------------

## Trabajos realizados

-   Se recolectaron los registros físicos y se digitalizaron manualmente
    utilizando **Google Sheets**, generando la primera versión del
    dataset.\
-   Se realizó un proceso de **limpieza, estandarización y procesamiento
    de datos** utilizando **Python** en **Google Colaboratory**,
    garantizando la calidad final del dataset.\
-   Se implementó una base de datos operativa en **Airtable**,
    permitiendo:
    -   Ingreso de datos mediante formularios claros y accesibles.\
    -   Actualización automática y disponibilidad online.\
    -   Funcionalidades de búsqueda, consulta y visualización.

Como cierre del proyecto, se documentó cada una de las etapas
realizadas. Además, se propone la integración futura de un **chatbot**
para registrar reclamos y consultas de los vecinos en una base de datos
secundaria, con el fin de automatizar y modernizar el proceso de
atención.
